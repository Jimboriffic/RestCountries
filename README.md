# RestCountries

Android app built with **Kotlin** and **Jetpack Compose** that lets you browse and search countries.

## Features
- Search countries by name
- Show/hide results using AnimatedVisibility
- Country details screen (if applicable)

## Tech Stack
- Kotlin
- Jetpack Compose
- AndroidX / Material (as used in the project)

## Getting Started

### Requirements
- Android Studio (latest stable recommended)
- Android SDK (API <minSdk>+)

### Run locally
1. Clone the repository  
   `git clone <REPO_URL>`
2. Open the project in Android Studio
3. Let Gradle sync
4. Run on an emulator or physical device
